# 1994 MTG World Championship (the first World Championship)

## Event Details

| Field | Value |
|-------|-------|
| Event | Magic: The Gathering World Championship 1994 (the inaugural Worlds) |
| Dates | August 18–21, 1994 (held during Gen Con) |
| Location | Milwaukee, Wisconsin, USA (Gen Con) |
| Players | Open entry, single-elimination (capped at 512 competitors) |
| Format | **Single sanctioned Constructed format of the era** — no "Type 2"/Standard existed yet (see Legal Sets below) |

## Notes on Accuracy

- **Dolan (1st)**: Complete 60-card main + 15-card sideboard, sourced from TappedOut and corroborated by the fabiensanglard writeup (which confirms Library of Alexandria, Stasis, Howling Mine, Icy Manipulator, and the "Keeper/Stasis" name). This is the famous deck running all five Moxen, Black Lotus, Ancestral Recall, Time Walk, Timetwister, Recall, dual lands (Savannah/Tropical Island/Tundra), and 4 Serra Angel. High confidence.
- **Lestrée (2nd)**: Complete list from mtgtop8 (event 9186, deck 252616), which records a **61-card** main deck + 15-card sideboard. This is the "Lestree Zoo" (Channel + Fireball, Kird Ape, Argothian Pixies, Lightning Bolt/Chain Lightning, with Serendib Efreet in the sideboard) — frequently called the first aggro deck. Good confidence; note some secondary reconstructions (MTGGoldfish 4812024 / cardsrealm) differ slightly (e.g. Mox Diamond vs Mox Emerald, Serendib main vs sideboard) — the mtgtop8 list is used here as the most consistent primary-style source.
- **Symens (3rd–4th)**: Complete 60 main / 15 sideboard from mtgtop8 (event 9186, deck 253151). High confidence.
- **de Foucaud (3rd–4th)**: Complete list from mtgtop8 (event 9186, deck 253152), which records a **66-card** main deck + 15-card sideboard. The over-60 main is faithful to the source (early-era decks were not always trimmed to exactly 60). High confidence.

**Engine note:** These decks use pre-modern cards (Power 9, original dual lands, Arabian Nights / Antiquities / Legends cards). Whether they load in the current mtg-forge-rs card database is unverified; this directory is historical provenance documentation, not a guarantee of engine compatibility.

## Legal Sets

In August 1994 there was **no rotating Constructed format**. "Type 2" / Standard did not yet exist (it was introduced in 1995). The first World Championship was played in what was simply *the* sanctioned Constructed format of the day — the same card pool later retroactively called **Type 1 / Vintage**. The legal card pool was essentially **every set printed to date**:

- **Limited Edition Alpha** (LEA, Aug 1993)
- **Limited Edition Beta** (LEB, Oct 1993)
- **Unlimited Edition** (2ED, Dec 1993)
- **Arabian Nights** (ARN, Dec 1993)
- **Antiquities** (ATQ, Mar 1994)
- **Legends** (LEG, Jun 1994)

**Boundary note:** **The Dark** released **August 1994**, essentially concurrent with the event, and is generally **not** counted as part of the 1994 Worlds pool (the contemporaneous decklists above contain no The Dark cards). **Revised Edition** (3ED, Apr 1994) was also in print; it is a reprint set, so it added no new cards beyond what Alpha/Beta/Unlimited/Arabian Nights/Antiquities already provided.

This era's rules also differed from modern Constructed:
- The 60-card-minimum deck, 15-card sideboard, and four-copies-of-any-nonbasic limit had just been codified and were in force at this event.
- **Ante cards and Shahrazad were banned** (the Duelist's Convocation, later the DCI, had issued the game's first Banned & Restricted list in early 1994).
- The **Power 9** and other broken cards (e.g. Channel, Mind Twist, Black Lotus, the Moxen, Ancestral Recall, Time Walk, Timetwister, Library of Alexandria) were **restricted to one copy each** — which is why every list above runs them as 1-ofs.

## Why this event is special

This is the **only** World Championship played in the all-cards / Type 1 (Vintage) pool. From **1995 onward** the World Championship always carried a mandatory **Type 2 (Standard)** main event (see the top-level `decks/championship/README.md` for the capture convention). The 1994 event therefore stands apart: it is the single main Constructed tournament of that year, captured here in full.
