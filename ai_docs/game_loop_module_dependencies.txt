Circular Dependency Prevention
================================================================================

Potential Problem: steps.rs and combat.rs both need priority_round()

Solutions:
1. Extract priority.rs FIRST (before steps/combat)
   - ✅ Cleanest approach
   - steps.rs and combat.rs import from priority.rs

2. Keep priority_round() in mod.rs initially
   - Extract steps/combat first
   - Move priority_round() later
   - ❌ More complex migration

Recommendation: Extract in this exact order:
  logging.rs → actions.rs → snapshot.rs → legacy.rs →
  priority.rs → steps.rs → combat.rs → mod.rs
