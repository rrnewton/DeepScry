mtg-forge-rs

An experiment in porting Magic: the Gathering to Rust — with tree-search AI in mind.

Built on the open-source Forge card library and rules engine, this project re-implements the core game in safe Rust with deterministic simulation, undo-log rewind, and a focus on AI research throughput. The engine runs natively on the server and as WebAssembly in the browser.

Play head-to-head against a friend over WebSocket, watch heuristic AI battle itself in your browser, or launch a fancy terminal-style UI — all from the same engine.
