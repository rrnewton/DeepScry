DeepScry

An experimental high-performance Magic: the Gathering engine in Rust — with tree-search AI in mind.

Card data is sourced from the open-source Forge project. The MTG rules engine and game simulation are an independent Rust reimplementation — not a port of Forge's Java rules code — built for deterministic simulation, undo-log rewind, and AI search. See PROJECT_VISION.md (original vision) for the design rationale. The engine runs natively on the server and as WebAssembly in the browser.

The single-thread engine simulates roughly 55K simple-deck games/sec (~165K turns/sec) on a modern desktop CPU (AMD Threadripper PRO 7975WX), and on the order of tens of thousands of turns/sec even on complex multi-color decks — reproducible with ./scripts/run_benchmark.sh.

Play head-to-head against a friend over WebSocket, watch heuristic AI battle itself in your browser, or launch a terminal-style Web TUI — all from the same engine.
