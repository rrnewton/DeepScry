# Description

## PER-CARD ASPECT ENUMERATION (2026-06-11_#3219(66619618), agent beads-enumerate-2025)

PURPOSE: an ACCURATE, SCRIPT-DERIVED checklist of every behavioral aspect each
unique card exhibits — the true surface area to verify, irrespective of current
compatibility. Each `- [ ]` is an UNCHECKED aspect; gameplay-evidence ticking is
a LATER phase (do NOT read these as WORKING). Derived by reading each card's
Forge-style script in cardsfolder/. Known BROKEN/status notes are preserved in
the STATUS NOTES section that follows; the checkboxes here describe what the card
is SUPPOSED to do per its printed text + script.

FORMAT per card:
  ### <Card> — <mana cost> <type line> [decks] (cardsfolder/<file>.txt)
  - [ ] <one aspect per printed ability / mode / keyword / notable interaction>
Legend: (A)=activated, (T)=triggered, (S)=static, (R)=replacement, (K)=keyword,
(cast)=cast-time/targeting, (cost)=cost/additional-cost, (int)=interaction/scaling.

Scope: 54 unique non-basic cards (union of decks/championship/2025/*.dck main+SB)
+ the 3 basic lands. All 54 resolve to a cardsfolder script (Roaring Furnace and
The Legend of Kuruk are DFC/Room combined files).

## Wave-8 Fixes (2026-06-12_#3384, branch: claude/compat-2025-wave8)

### FIXED: Eddymurk Crab ReduceCost Amount$X variable SVar (3-part fix)

Root cause: `S:Mode$ ReduceCost | ValidCard$ Card.Self | Amount$ X | EffectZone$ All`
with `SVar:X:Count$ValidGraveyard Instant.YouOwn,Sorcery.YouOwn` had three bugs:

1. **Amount$X not evaluated as dynamic**: `value.parse::<u8>().unwrap_or(0)` returns 0
   for "X". Fixed by calling `DynamicAmount::parse(value, &svars)` and storing
   `CostReductionAmount::Dynamic(CountExpression::ValidGraveyard{...})`.

2. **Card.Self → AllSpells matched wrong path**: `ValidCard$ Card.Self` was mapped
   to `CostReductionTarget::AllSpells`, but the battlefield loop only checked
   permanents already in play — the spell itself (still in hand) was never
   checked. Fixed by adding `CostReductionTarget::SelfCard` and a dedicated
   self-ReduceCost pass in `calculate_effective_cost` that reads the card's own
   static abilities before the battlefield scan (CR 601.2e).

3. **Comma-separated type filter not supported**: `count_cards_matching_filter`
   couldn't handle `"Instant.YouOwn,Sorcery.YouOwn"`. Fixed by adding a
   comma-split union/dedup path at the top of the function.

Evidence: `tests/eddymurk_crab_self_reduce_cost_e2e.sh` PASSES — P0 casts a
7-mana creature for only 2 Islands (5 instants/sorceries in graveyard → 5
generic mana reduction → cost 5UU becomes 0UU).
