# Description

Card Compatibility: Red Elemental Blast

Card file: cardsfolder/r/red_elemental_blast.txt
Deck: 06_troll_disk_daniellebrunazzo (mtg-562, sideboard)
Test puzzle: test_puzzles/red_elemental_blast_destroy_blue.pzl

Card text:
  {R}  Instant
  Choose one —
  • Counter target blue spell.
  • Destroy target blue permanent.

Script:
  A:SP$ Charm | Choices$ DBCounter,DBDestroy
  SVar:DBCounter:DB$ Counter | TargetType$ Spell | ValidTgts$ Card.Blue
  SVar:DBDestroy:DB$ Destroy | ValidTgts$ Permanent.Blue

Findings (2026-05-29_#2470(be2f61b4), wave10):

1. [x] Parses as {R} Instant with a Charm ModalChoice of two modes.
2. [BROKEN -> FIXED] Both modes now enforce the per-mode color restriction
   (was mtg-af24s): Destroy mode -> Permanent.Blue, Counter mode -> Card.Blue.
   Previously REBL could destroy/counter any color (illegal targeting,
   CR 115.4); a repro destroyed the caster's own red Mountain.
3. [x] Destroy mode targets only BLUE permanents (verified: destroys opponent's
   Phantom Monster, not own red Mountain).
4. [x] Counter mode restricted to BLUE spells (parser-verified required_color;
   targeting filters stack spells by color).
