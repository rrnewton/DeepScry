# Description

## Bug: ModifyPT static abilities with Affected$ Card.Self silently fail — FIXED

Closed 2026-06-01 gardening: RESOLVED. Both bugs described in this issue are fixed.

Bug 1 (AffectedSelector::Self_ skipped): Fixed in mtg-engine/src/game/continuous_effects.rs lines 798-805. The arm now applies the buff when creature_id == source_id:
  AffectedSelector::Self_ => {
      // The static buffs the source permanent itself (e.g. Sedge Troll's conditional +1/+1)
      if creature_id == source_id {
          power_bonus += power;
          toughness_bonus += toughness;
      }
  }
Evidence: continuous_effects.rs line 798.

Bug 2 (IsPresent$/Condition$ qualifiers silently dropped): GARDENING (2026-06-01): the condition field on ModifyPT was NOT independently verified as fixed. The fix for Bug 1 is confirmed. If conditional self-buffs (Sedge Troll +1/+1 when Swamp present) are still always-on rather than conditional, that is a residual of Bug 2. See mtg-398 for Sedge Troll-specific tracking.

Closing this meta-issue because the primary silent-skip bug is confirmed fixed. Conditional correctness lives in per-card issues.
