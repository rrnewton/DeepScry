# Description

Test all behavioral aspects of Power Sink in MTG Forge-rs.

Card: cardsfolder/p/power_sink.txt
Set: LEA (Alpha)
Deck: thedeck Peterschnidrig (mtg-413, sideboard)

Card text:
  Power Sink {X}{U} Instant
  Counter target spell unless its controller pays {X}. If that player
  doesn't, they tap all lands with mana abilities they control and
  lose all unspent mana.

Script:
  A:SP$ Counter | UnlessCost$ X | TargetType$ Spell | ValidTgts$ Card
   | SubAbility$ TapLands | UnlessResolveSubs$ WhenNotPaid
  SVar:TapLands:DB$ TapAll | ValidCards$ Land.hasManaAbility
   | Defined$ TargetedController | TapperController$ True | SubAbility$ ManaLose
  SVar:ManaLose:DB$ DrainMana | Defined$ TargetedController
  SVar:X:Count$xPaid

Findings (2026-05-30_#2530(199b91e1), compat-wave13-thedeck):

1. [x] Parses as {X}{U} Instant ({X} cost, blue pip). Unit test
       test_card_compat_power_sink.
2. [x] SP$ Counter UnlessCost$ X: caster picks X, targets a spell,
       opponent gets pay-{X} prompt; when not paid the SubAbility chain runs.
3. [x] TapAll fires (taps opponent's mana lands).
4. [WORKING] DB$ DrainMana ("lose all unspent mana") IMPLEMENTED. Was
       previously ApiType::Unknown("DrainMana") → logged no-op. Now a
       concrete ApiType::DrainMana → Effect::DrainMana { player }. The
       player sentinel resolves Defined$ TargetedController to the
       countered spell's controller (via last_resolved_target). Execution
       empties that player's mana pool (CR 106.4 / 500.4) and logs
       "<player> loses all unspent mana (N drained)".
5. [N/A] Mana production / triggers / static / replacement — none.

GENERAL FIX (lifts other DrainMana cards): added ApiType::DrainMana
(ability_parser.rs), Effect::DrainMana { player } (core/effects.rs),
converter arm (effect_converter.rs), sentinel resolution
(actions/mod.rs resolve_effect_target), and execution
(actions/mod.rs execute_effect: empty_mana_pool + gamelog). Also wired
the puzzle loader's previously-parsed-but-unapplied 'manapool=' field
(puzzle/loader.rs) so tests can pre-float mana.

test_drain_mana_empties_pool.)
