# Description

Test all behavioral aspects of Sedge Troll in MTG Forge-rs.

Card: cardsfolder/s/sedge_troll.txt
Set: LEG
Deck: rogue_rogerbrand (mtg-387)
Test puzzle: test_puzzles/sedge_troll_swamp_boost.pzl

Card text:
  Sedge Troll {2}{R}, 2/2 Creature - Troll
  +1/+1 as long as you control a Swamp.
  {B}: Regenerate Sedge Troll.

Findings (2026-05-12, compat2):

1. [x] Parses as 2/2 Creature - Troll, cost {2}{R}
2. [x] Activated ability {B}: Regenerate parses and works
3. [x] Static ability ModifyPT (+1/+1) parses correctly
4. [BROKEN] Static +1/+1 NOT applied at runtime — always 2/2.
   - Bug 1: AffectedSelector::Self_ skipped in calculate_modifypt_effects.
   - Bug 2: IsPresent$ Swamp.YouCtrl condition silently dropped (ModifyPT had no condition field).

Findings (2026-05-29_#2430(048328c1), compat-oldschool-wave1):

FIXED — CARD STATUS: WORKING.

Both root causes addressed:
- Added StaticCondition::ControlsPresent { filter, zone, min_count } so a
  Continuous static can carry an IsPresent$ condition (reuses the same
  is_present/present_zone/present_compare_min parsing already used for
  ReduceCost, and the existing count_cards_matching_filter at runtime).
- ModifyPT now has a `condition: Option<StaticCondition>` field (mirroring
  GrantKeyword). Parser populates it from IsPresent$ (or Condition$).
- Runtime: calculate_modifypt_effects now (a) applies Affected$ Card.Self to
  the source itself and (b) gates the bonus on a shared static_condition_met()
  helper (also used by GrantKeyword — DRY).

1. [x] Static +1/+1 applied WHEN a Swamp is controlled (3/3).
2. [x] Static +1/+1 NOT applied when no Swamp (stays 2/2) — condition gating verified.
3. [x] {B}: Regenerate still works.

Reproducer:

```sh
./target/release/mtg tui --start-state test_puzzles/sedge_troll_swamp_boost.pzl \
  --p1=heuristic --p2=zero --stop-on-choice=6 --json --seed 42 --verbosity 3
```

Expected log evidence (with a Swamp present):

```

  Sedge Troll (3) deals 3 damage to Player 2 (life: 17)
```

Negative case (no Swamp on battlefield) shows `Sedge Troll 2/2`.
