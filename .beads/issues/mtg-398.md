# Description

Test all behavioral aspects of Sedge Troll in MTG Forge-rs.

Card: cardsfolder/s/sedge_troll.txt
Set: LEG
Deck: rogue_rogerbrand (mtg-387)

Card text:
  Sedge Troll {2}{R}, 2/2 Creature - Troll
  +1/+1 as long as you control a Swamp.
  {B}: Regenerate Sedge Troll.

Findings (2026-05-12, compat2):

1. [x] Parses as 2/2 Creature - Troll, cost {2}{R}
2. [x] Activated ability {B}: Regenerate parses and works
   - Gameplay evidence: 'Sedge Troll activates ability: Regenerate Sedge Troll' then 'Sedge Troll (3) gains a regeneration shield'
3. [x] Static ability ModifyPT (+1/+1) parses correctly
4. [BROKEN] Static +1/+1 NOT applied at runtime — Sedge Troll always shows 2/2 even when controlling a Swamp.
   - Bug 1: AffectedSelector::Self_ is intentionally skipped in calculate_modifypt_effects (continuous_effects.rs:788), comment says 'Equipment affecting itself' — wrong for creatures with self-buff statics.
   - Bug 2: IsPresent\$ Swamp.YouCtrl conditional qualifier is silently dropped by static-ability parser (card.rs:3640 — ModifyPT struct has no condition field; only GrantKeyword does).
   - Affects: many other 93-94 cards, plus modern conditional self-buffers (Anurid Barkripper, Aerial Engineer, Akki Underling, Akiri Line-Slinger, Adelbert Steiner, ...). All silently fail closed (always base P/T).

Reproducer:

```sh
cargo build --release
cat > /tmp/sedge_swamp.pzl <<'P'
[metadata]
Name:Sedge Troll Swamp Boost
Goal:Win
Turns:5
Difficulty:Easy

[state]
turn=2

activephase=MAIN1
p0life=20
p0hand=
p0library=Mountain
p0graveyard=
p0battlefield=Sedge Troll; Swamp; Mountain
p0exile=
p1life=20
p1hand=
p1library=Mountain
p1graveyard=
p1battlefield=
p1exile=
P
./target/release/mtg tui --start-state /tmp/sedge_swamp.pzl --p1=zero --p2=zero --stop-on-choice=2 --seed 42 --verbosity verbose
```

Expected: Sedge Troll 3/3 (with Swamp present)
Actual:   Sedge Troll 2/2

Unit test: test_card_compat_sedge_troll in mtg-engine/src/game/actions/tests/effects.rs pins the parser shape and documents the runtime gap.

CARD STATUS: PARTIAL — regenerate works; conditional +1/+1 boost is silently broken.
