# Description

Test all behavioral aspects of Wrath of God in MTG Forge-rs.

Card: cardsfolder/w/wrath_of_god.txt
Set: LEA (Alpha)
Deck: thedeck Peterschnidrig (mtg-413, sideboard)

Card text:
  Wrath of God {2}{W}{W}, Sorcery
  Destroy all creatures. They can't be regenerated.

Script:
  A:SP$ DestroyAll | ValidCards$ Creature | NoRegen$ True

Findings (2026-05-29_#2470(be2f61b4), compat-oldschool-wave9):

1. [x] Parses as {2}{W}{W} Sorcery (CardType::Sorcery, generic=2,
       white=2).
2. [x] SP$ DestroyAll with ValidCards$ Creature and NoRegen$ True is
       converted to Effect::DestroyAll and resolves correctly.
3. [x] Casting Wrath of God destroys all creatures regardless of
       controller, sending each to its owner's graveyard.
4. [x] The NoRegen$ True flag is honored: the engine's
       Effect::DestroyAll path takes the `no_regenerate` branch and
       emits the "(can't be regenerated)" log marker. The destroy
       loop in mod.rs has explicit "if has_regen_shield &&
       !*no_regenerate" handling (mtg-engine/src/game/actions/mod.rs).
5. [N/A] Mana abilities — sorcery has none.
6. [N/A] Triggered / replacement / static abilities — only the spell
       ability is present.
