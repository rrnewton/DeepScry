# Description

Files: agentplay/agent_game.py or agentplay/agent_prompt.py (new module)

Action: Design the prompt template that gets sent to claude -p for each choice:
1. Include current game state from snapshot (battlefield, hands, life totals, stack, phase)
2. List available choices with numbers: [0] pass, [1] play Forest, etc.
3. Include game history / recent log lines for context
4. Include any goal directive (for Phase 2)
5. Instruct Claude to output ONLY a single numeric choice as the first line, followed by reasoning
6. Parse the response: extract numeric choice (first line/number), capture reasoning text

The prompt should be clear enough that Claude can make intelligent MTG decisions. Include:
- Current phase/step
- Active player's hand, battlefield, graveyard  
- Opponent's visible state (battlefield, graveyard, hand size)
- Stack contents
- Life totals
- Available mana
- The choice list

Verify:
- Prompt produces valid output format from claude -p
- Numeric choice is correctly parsed from response
- Reasoning is captured for the enriched log
