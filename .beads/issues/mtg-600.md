# Description

Found fixing mtg-589. Hypnotic Specter trigger (SVar:TrigDiscard:DB Discard | Defined TriggeredTarget | NumCards 1 | Mode Random) has two pre-existing gaps in effect_converter.rs (ApiType::Discard): (1) Defined TriggeredTarget unhandled -> resolves to spell controller instead of the damaged opponent (Brand discards instead of Roger); (2) Mode Random dropped -> discard is deterministic not random. The mtg-589 fix made forced discards network-deterministic (lowest CardId) which removes the desync and is outcome-equivalent, but a correct fix needs server-authoritative random selection synced to shadows (analogous to library_search_result) plus parsing TriggeredTarget + Mode Random. Repro: rogerbrand seed 3 gamelog 'Brand discards Swamp' after Hypnotic Specter damages Roger. Related: mtg-589, mtg-212, mtg-259.
