# Description

Test all behavioral aspects of Balance in MTG Forge-rs for the 1994 Old School playtest goal.

Card file: cardsfolder/b/balance.txt
Deck: 06_troll_disk_daniellebrunazzo (mtg-562, sideboard)
Test puzzle: test_puzzles/balance_spell_sacrifice.pzl

Card text:
  {1}{W}  Sorcery
  Each player chooses a number of lands they control equal to the number of
  lands controlled by the player who controls the fewest, then sacrifices the
  rest. Players discard cards and sacrifice creatures the same way.

Script:
  A:SP$ Balance | Valid$ Land | SubAbility$ BalanceHands
  SVar:BalanceHands:DB$ Balance | Zone$ Hand | SubAbility$ BalanceCreatures
  SVar:BalanceCreatures:DB$ Balance | Valid$ Creature

Findings (2026-05-29_#2470(be2f61b4), wave10):

1. [x] Parses as {1}{W} Sorcery; SP$ Balance ability -> Effect::Balance.
2. [x] Lands equalize: each player sacrifices down to the fewest-lands count.
3. [x] Hands equalize: each player discards down to the smallest hand size.
4. [x] Creatures equalize: each player sacrifices down to the fewest-creatures
   count.
5. [x] Full SubAbility chain (Land -> Hand -> Creature) runs in order; no
   sentinels / silent drops in the log.

No engine change required — the Balance effect + SubAbility chain already
worked. This pass adds the regression coverage that was missing (parser-shape
unit test + wired e2e).

Reproducer:

```sh
./target/release/mtg tui --start-state test_puzzles/balance_spell_sacrifice.pzl \
  --p1=fixed --p2=zero --p1-fixed-inputs="cast Balance;*;*;*;*;*;*;*" \
  --seed 42 --verbosity 3
```
