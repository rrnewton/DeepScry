# Description

## Context

Commit b75ed4cb fixed Bazaar of Baghdad's discard-3 in tui_game.html by adding a multi-card staging UI in the WASM TUI. The fix is bespoke to discard.

## Pattern
Many MTG effects require choosing N>1 entities from a set:
- Discard N: Hymn to Tourach (opp discards 2), Mind Twist (X), Bazaar of Baghdad (3)
- Sacrifice N: Smallpox, Plague of Vermin
- Tap N for cost: Crovax-style 'tap two creatures'
- Choose N targets: Arc Lightning, Forked Bolt
- Mill N to grave + select: Surveil/Scry-then-act variants

## Goal
Refactor the staging buffer added in fancy_tui.rs (around the discard menu) into a generic multi-pick component so any choice with NumCards>1 (or NumTargets>1) can show: live count, deselect, confirm.

## Files
- mtg-engine/src/wasm/fancy_tui.rs (multi-pick staging logic)
- mtg-engine/src/game/state.rs (choice context surface)

## Verify
Hymn to Tourach, Mind Twist (X=2), Forked Bolt — each presents a working staged pick UI in tui_game.html and game.html.
