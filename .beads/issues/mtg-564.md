# Description

BROKEN→WORKING (verified 2026-05-30_#2530(199b91e1), compat-wave13-thedeck): target player discards X cards at random.

Card script: cardsfolder/m/mind_twist.txt
Effect: SP$ Discard | ValidTgts$ Player | NumCards$ X | Mode$ Random

USER BUG REPORT (fix-gameplay-bugs-4pack): 'In testing the jeskai aggro old school decks, the opponent cast mind twist on me with X=8 and I did not discard ANY cards.'

ROOT CAUSE (historical): effect_converter.rs ApiType::Discard resolved ValidTgts$ Player to the caster (card_owner) instead of the targeted opponent.

FIX (already landed in a prior wave, mtg-521): effect_converter.rs now maps (_, Some(vt)) if vt.contains("Player") => PlayerId::target_opponent(), and actions/mod.rs resolves the target_opponent sentinel to opponent_id for both DiscardCards and DiscardCardsXPaid. CR 116.2c / 601: targeted spells require a chosen target; approximated as the sole opponent in 2-player games.

VERIFICATION (this wave): drove a real game — opponent's hand (Player 2) is the one that discards.

Reproducer:

```sh
./target/release/mtg tui --p1=fixed --p2=zero \
  --p1-fixed-inputs='cast Mind Twist;2;*;*' --seed 42 --verbosity 3
## (with p0hand=Mind Twist, p1hand=Plains;Plains;Plains)
```

Expected log:
```
Player 1 casts Mind Twist (3) (putting on stack)
  → X = 2
Player 2 discards Plains
Player 2 discards Plains
```

Unit test: test_card_compat_mind_twist (mtg-engine/src/game/actions/tests/effects.rs)
E2E tests: tests/mind_twist_discard_e2e.sh, tests/mind_twist_x_discards_opponent_e2e.sh

MTG Rules Review — Verdict: PASS
1. Correct rule: CR 116.2c/601 targeting; opponent discards at random (CR 701.8). 
2-5. Reveal/info-hiding/authority/sync: random discard from opponent's hand uses the engine's seeded RNG and does not leak hand contents to the caster's controller (info-independent).
6. Real fix (converter target default), not a card hack.
7. Generalizes to all ValidTgts$ Player discard cards.
