# Description

Test all behavioral aspects of Control Magic in MTG Forge-rs for the 1994 Old School playtest goal.

Card file: cardsfolder/c/control_magic.txt
Deck: 06_troll_disk_daniellebrunazzo (mtg-562)
Test puzzle: test_puzzles/control_magic_steal_revert.pzl

Card text:
  {2}{U}{U}  Enchantment - Aura
  Enchant creature
  You control enchanted creature.

Script:
  K:Enchant:Creature
  S:Mode$ Continuous | Affected$ Card.EnchantedBy | GainControl$ You

Findings (2026-05-29_#2470(be2f61b4), wave10):

1. [x] Parses as Enchantment Aura, cost {2}{U}{U}, K:Enchant:Creature.
2. [x] `S:Mode$ Continuous | GainControl$ You` now parses into
   StaticAbility::GainControl (previously the GainControl$ param was
   silently dropped by parse_static_abilities -> the Aura attached but
   never transferred control).
3. [x] Cast on opponent's creature transfers control (CR 613.2 layer-2
   control change), applied continuously in
   GameState::recompute_aura_control() during the SBA loop.
4. [x] Control reverts to the owner the moment the Aura leaves the
   battlefield (destroyed by Disenchant, host dies, bounce) — CR 704.5.
   Tracked per-permanent via the new Card.control_from_aura field so the
   revert does NOT clobber control gained by other mechanisms (Animate
   Dead's one-shot GainControl$ True, Threaten's AB$ GainControl).

GENERALIZED to the whole control-stealing-Aura bug class: 56 Aura cards
use `S:Mode$ Continuous | GainControl$ You` (Mind Control, Persuasion,
Enslave, Confiscate, Volition Reins, Steal Artifact, ...). All are fixed
by the same StaticAbility::GainControl + recompute_aura_control path.

Reproducer:

```sh
./target/release/mtg tui --start-state test_puzzles/control_magic_steal_revert.pzl \
  --p1=fixed --p2=zero \
  --p1-fixed-inputs="cast Control Magic;cast Disenchant;Control Magic;*" \
  --seed 3 --verbosity 3
```

Expected log evidence:

```
Control Magic enchants Savannah Lions
Savannah Lions comes under Player 1's control
Disenchant (4) destroys Control Magic (3)
Savannah Lions returns to Player 2's control
```
