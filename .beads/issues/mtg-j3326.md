# Description

Root-cause bug found in the 2025 World Championship survey (mtg-881, branch claude/compat-2025-survey).

STAMP: 2026-06-10_#3175(c6dbd34f)

== Symptom ==
A 200-game tourney (seed 7) across decks/championship/2025/*.dck emits 135x:
  [WARN  mtg_engine::game::actions] DealDamageXPaid reached execute_effect without resolution, treating as 0 damage
Attribution: the warning is emitted ONLY by decks 02 (Shibata) and 04 (Henry) — the two decks containing Torch the Tower. Decks 01 and 03 (which contain Combustion Technique but NOT Torch) emit ZERO. So this is NOT the Combustion Technique X-bug (that one, mtg-831, was genuinely fixed). It is Torch the Tower.

== Card ==
cardsfolder/t/torch_the_tower.txt (Torch the Tower, R Instant, K:Bargain; 1x main deck 02, 4x main deck 04, sideboard 01)
  A:SP$ DealDamage | ValidTgts$ Creature,Planeswalker | NumDmg$ X | RememberDamaged$ True | ReplaceDyingDefined$ Remembered ...
  SVar:X:Count$Bargain.3.2
"Torch the Tower deals 2 damage to target creature or planeswalker. If this spell was bargained, instead it deals 3 damage and you scry 1."

== Root cause ==
NumDmg$ X with SVar:X:Count$Bargain.3.2 (the conditional-count form "N if Bargained else M") does not resolve to a scalar at execute time; it reaches execute_effect as DealDamageXPaid with no resolved value, so the engine logs the WARN and substitutes 0 damage. Same CLASS as the Combustion Technique Count$ X-damage bug (mtg-820 / DealDamageDynamic) but a DIFFERENT Count form — Count$Bargain.N.M is still unhandled.

== Empirical evidence ==
debug/compat2025/v3_02_shibata.log (gitignored, branch claude/compat-2025-survey), lines ~505-513:
  AI-Heuristic2 casts Torch the Tower (98) -> targeting Gran-Gran (118)
  Torch the Tower (98) resolves
  [WARN ...] DealDamageXPaid reached execute_effect without resolution, treating as 0 damage
Gran-Gran (a 1/2) SURVIVED the "torch" and kept attacking for several more turns (lines 180-397 same log). Expected: 2 damage (3 if bargained) kills the 1/2 and exiles it.

== Reproducer ==
```sh
./target/release/mtg tourney decks/championship/2025/02_shibata_izzet_lessons.dck --games 12 --seed 7 2>&1 | grep -c "DealDamageXPaid"
```
Expected stdout (current, BROKEN): nonzero count (observed 9). After fix: 0.

== Fix shape ==
Implement Count$Bargain.N.M conditional-count resolution so NumDmg$ X yields the correct scalar (M base, N if the spell was bargained) BEFORE execute_effect, on the path that already resolves Count$ValidGraveyard for Combustion Technique (DealDamageDynamic). Game-logic change -> requires MTG rules review. DO NOT START until the mtg-245 execute_effect dispatch-table refactor lands (it owns this path; a fix now would collide).

== Affected cards ==
Torch the Tower (this survey). Any card using NumDmg$ X with a Count$Bargain (or similar two-arm conditional Count) form.

Rolls up under mtg-881 / 2025 backlog. CARD STATUS for Torch the Tower (mtg-863): BROKEN.
