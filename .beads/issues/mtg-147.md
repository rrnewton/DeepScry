# Description

Track implementation of missing Affected$ selectors in static ability parsing.

## Overview

Random deck tournament (2025-11-23_#894) revealed many unhandled `Affected$` selectors in static ability parsing. These cause cards with complex static abilities to silently fail.

## Progress Summary (as of 2025-12-04_#1125)
