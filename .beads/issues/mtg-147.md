# Description

Random deck tournament (2025-11-23_#894) revealed many unhandled \`Affected$\` selectors in static ability parsing. These cause cards with complex static abilities to silently fail.

## How to Check Progress

Run the stats command and count warnings:

\`\`\`bash
## Total warning count
cargo run --release --bin mtg -- stats 2>&1 | grep "Warning" | wc -l

## See specific unknown Affected$ selectors and their counts
cargo run --release --bin mtg -- stats 2>&1 | grep "Unknown Affected" | \\
  sed "s/.*selector '\([^']*\)'.*/\1/" | sort | uniq -c | sort -rn | head -20

## See all warning types
cargo run --release --bin mtg -- stats 2>&1 | grep "Warning:" | \\
  sed 's/Warning: //' | cut -d' ' -f1-5 | sort | uniq -c | sort -rn | head -20
\`\`\`

The \`mtg stats\` command loads all 31,545 cards and emits a warning for each unhandled pattern.
