# Description

The WASM export (mtg export-wasm) doesn't include decks/tmnt/*.dck by default.

The default deck_globs in main.rs only include:
- decks/old_school/**/*.dck
- decks/old_school2/**/*.dck  
- decks/booster_draft/**/*.dck

Need to add decks/tmnt/**/*.dck to the default list.

Alternatively, the user can pass --deck-globs manually, but adding it to defaults
ensures TMNT decks are available in the web UI.
