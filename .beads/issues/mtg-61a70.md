# Description

Game hangs after choice_request seq=127 when server sends available_count=0 but abilities array contains 1 ability. After P1 casts Cunning Maneuver (PumpCreature + CreateToken), action_count jumps 584→587 (3 intermediate actions). WASM local controller validation fails: server_abs.len() (1) \!= available.len() (0). Root cause: WASM client local state hasn't processed intermediate actions between choices. Key files: local_controller.rs, fancy_tui.rs, priority.rs.
