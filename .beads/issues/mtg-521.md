# Description

Test all behavioral aspects of Mind Twist in MTG Forge-rs for the 1994
Old School playtest goal.

Card file: cardsfolder/m/mind_twist.txt

Card text:
  {X}{B}  Sorcery
  Target player discards X cards at random.

Script:
  A:SP$ Discard | ValidTgts$ Player | NumCards$ X | Mode$ Random
  SVar:X:Count$xPaid

Decks containing this card (1994 Old School playtest):
- 06_jeskai_aggro_joseantonioprieto (mtg-561)

== Findings (2026-05-29_#2462(132ce6cc)) ==

1. [x] Parses with cost {X}{B} (mana_cost.black=1, generic=0; X via Cost::Generic
   accumulating per pip). Type: Sorcery.
2. [x] Parser produces Effect::DiscardCardsXPaid with PlayerId::target_opponent()
   sentinel (per ValidTgts$ Player; see test_card_compat_mind_twist).
3. [x] Cast for X=1 with {B} + {1} (Mox Jet + Badlands) — pays correctly,
   X resolves to 1 at resolve_x_paid_effect() time.
4. [x] Resolution discards exactly X cards from the target opponent.
   "Mode$ Random" picks the random card; for seed=42 this is Plains.
   Each discard emits "Fixed2 discards <CardName>" — non-sentinel.
5. [x] FIXED IN THIS COMMIT (mtg-521): post-resolution summary log now shows
   the resolved X count and the resolved opponent name, not the literal
   sentinel strings. Before: "Mind Twist (0) causes target opponent to
   discard X card(s)". After: "Mind Twist (0) causes Fixed2 to discard 1
   card(s)".
   Root cause: log_effect_execution() in mtg-engine/src/game/game_loop/
   logging.rs is called with the pre-resolution Effect::DiscardCardsXPaid;
   the post-resolution remap loop in priority.rs didn't convert
   DiscardCardsXPaid -> DiscardCards{count: x_paid}, and didn't resolve
   PlayerId::target_opponent() to the actual opponent id, so the formatter
   showed the sentinel string. Fixed by capturing card.x_paid alongside
   card_effects and adding XPaid -> concrete-variant mappings plus a
   resolve_log_player() helper.
6. [x] Caster (Fixed1) does NOT discard — the target_opponent() sentinel
   round-trips correctly to the opponent player only (mtg-564 regression).
