# Description

Currently different controllers print choice logs inconsistently.

**Example from RandomController:**
```
Alice chose 0 - Play land: Mountain
```

**Goal:**
Every choice made by any controller should use a standard format with a `<Choice>` tag:
```
<Choice> Alice chose 0 - Play land: Mountain
```

**Benefits:**
- Can grep for `<Choice>` to count total choices in a game
- Consistent logging regardless of controller type (Random, AI, TUI, Fixed, etc.)
- Easier to analyze game logs and decision points

**Implementation:**
- Find all places where controllers log choices
- Add shared helper function or method for logging choices
- Update all controllers to use the standard format
- Verify with example game logs

**Related controllers:**
- RandomController
- HeuristicController  
- ZeroController
- FixedScriptController
- RichInputController (TUI)
- ReplayController
