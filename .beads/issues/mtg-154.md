# Description

Display actual stack contents in the Stack pane instead of always showing "(Stack empty)".

## Implementation (completed in commit f7d4e74)

1. Added GameStateView::stack() method to expose stack contents
2. Updated draw_stack() to display cards with position indicators [BOT], [TOP]
3. Highlighted top of stack in yellow/bold (resolves next)
4. Used cyan for position indicators

## Visual Design

Shows cards in order from bottom to top with clear position indicators:
- [BOT] for bottom card
- [   ] for middle cards
- [TOP] for top card (highlighted yellow/bold, resolves next)

## Benefits

- Players can see stack state without checking logs
- Clear indication of resolution order
- Better understanding during spell interactions
- Matches Java Forge GUI functionality
