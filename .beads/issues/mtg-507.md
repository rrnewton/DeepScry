# Description

Card Compatibility: Gloom (cardsfolder/g/gloom.txt) — mtg-507

Decks containing this card (1994 Old School playtest):
- 05_mono_black_rogerbrand (mtg-560) sideboard

Per .claude/skills/compatibility_tracking/SKILL.md.

Card text:
  {2}{B} Enchantment
  White spells cost {3} more to cast.
  Activated abilities of white enchantments cost {3} more to activate.

Script:
  S:Mode$ RaiseCost | ValidCard$ Card.White | Type$ Spell | Amount$ 3
  S:Mode$ RaiseCost | ValidCard$ Enchantment.White | Type$ Ability | Amount$ 3 | AffectedZone$ Battlefield

== Findings (2026-06-01_#2587(1faf5ae6), compat-monoblack-v2) ==

1. [WORKING] Parses as {2}{B} Enchantment with TWO RaiseCost static abilities.
2. [WORKING] White spells cost {3} more to cast when Gloom is on the battlefield.
   Verified by RUST_LOG=debug showing:
     RaiseCost from Gloom: White spells cost {3} more to cast. (increasing generic by 3, was 3, now 6)
   Serra Angel (normally 3WW) effectively costs 6WW when Gloom is on the field.
3. [WORKING] Heuristic AI now casts Gloom when the opponent controls white permanents.
   Previously the heuristic returned false from should_cast_global_enchantment because
   it only checked for ModifyPT/GrantKeyword/triggers; it didn't handle RaiseCost.
   Fix: added RaiseCost arm in heuristic_controller.rs::should_cast_global_enchantment
   that checks whether the opponent controls permanents matching the RaiseCost target.

NOTE: The second S: line (activated abilities of white enchantments cost {3} more)
is parsed but not exercised in this deck test (no white enchantments with activated
abilities in the standard 1994 old school matchup). The RaiseCost machinery
handles it via the same code path.

Reproducer:

```sh
RUST_LOG=debug ./target/release/mtg tui \
  /tmp/thedeck_gloom.dck \
  /tmp/monoblack_gloom.dck \
  --p1-draw "Serra Angel;Plains;Plains;Plains;Plains;Plains" \
  --p2-draw "Gloom;Swamp;Swamp;Swamp;Swamp;Swamp" \
  --p1=heuristic --p2=heuristic \
  --seed 42 --verbosity 2 2>&1 | grep "increasing generic"
```

Expected log evidence:

```
RaiseCost from Gloom: White spells cost {3} more to cast. (increasing generic by 3, was 3, now 6)
```

E2E test:     tests/gloom_raises_white_spell_cost_e2e.sh
