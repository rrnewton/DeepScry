1. [x] Parses as {1}{G} Enchantment.
2. [PARTIAL] Draw-step trigger now RECOGNISED (was silently dropped): Phase$ Draw maps to TriggerEvent::BeginningOfDraw and fires from the battlefield on the controller's draw step after the mandatory draw (CR 504.1 + 603.3). VERIFIED via unit test (test_card_compat_sylvan_library asserts the BeginningOfDraw trigger exists).
3. [BROKEN] The trigger's effect chain (the actual Sylvan Library text) is NOT implemented, so the trigger fires as a no-op: no extra draw, no choose-two, no pay-4-life-or-return. Requires AB$ ChooseCard + Cost$ Draw<N/You>, per-card DrawnThisTurn tracking, DB$ RepeatEach, and a per-card UnlessCost$ PayLife<4> hand->library-top move. Filed as: mtg-c4p8f.

Net effect: Sylvan Library is currently strictly WEAKER than printed (draws nothing extra) but is no longer a silent parser drop or crash. It is safe to include in a deck.
