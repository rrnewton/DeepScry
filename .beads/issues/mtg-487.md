# Description

Test all behavioral aspects of Blue Elemental Blast in MTG Forge-rs.

Card: cardsfolder/b/blue_elemental_blast.txt
Set: LEA (Alpha)
Deck: thedeck Peterschnidrig (mtg-413, sideboard)

Card text:
  Blue Elemental Blast {U} Instant
  Choose one —
   • Counter target red spell.
   • Destroy target red permanent.

Script:
  A:SP$ Charm | Choices$ DBCounter,DBDestroy
  SVar:DBCounter:DB$ Counter | TargetType$ Spell | ValidTgts$ Card.Red
  SVar:DBDestroy:DB$ Destroy | ValidTgts$ Permanent.Red

Findings (2026-05-29_#2473(2c742260), compat-oldschool-wave9):

1. [x] Parses as {U} Instant; SP$ Charm with Choices$ DBCounter,DBDestroy
       resolves both SVars into the expected modal-effect list.
2. [x] Mode 1 (counter red spell) — verified by puzzle
       test_puzzles/blue_elemental_blast_counter.pzl: P1 casts
       Lightning Bolt (a red instant); P2 responds with Blue Elemental
       Blast, selects "Counter target red spell"; the Bolt is countered.
3. [x] Mode 2 (destroy red permanent) — verified by puzzle
       test_puzzles/blue_elemental_blast_destroy.pzl: P2 casts BEB on
       P1's Mountain (a red permanent); selects "Destroy target red
       permanent"; the Mountain is destroyed and graveyarded.
4. [x] Red filter honored: only red spells/permanents are offered as
       legal targets (heuristic legal-target enumeration in the
       targeting layer).
5. [N/A] Mana abilities — instant with no mana ability.
6. [N/A] Triggered / static / replacement abilities — none printed.

Reproducer (counter mode):

./target/release/mtg tui \
