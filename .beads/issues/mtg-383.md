# Description

Track card compatibility for Limited Edition Alpha cards used in MTG Forge-rs decks (initially the rogue_rogerbrand 93/94 deck).

Cards from LEA used in decks:
- Sengir Vampire (4x in rogue_rogerbrand)
- Shivan Dragon (4x)
- Sedge Troll (3x)
- Chaos Orb (1x)
- Mox Pearl (1x)
- Mox Jet (1x)
- Mox Ruby (1x)
- Mox Emerald (1x)
- Swords to Plowshares (4x)
- Disenchant (4x)
- Wheel of Fortune (1x)
- Animate Dead (4x)
- Badlands (4x)
- Scrubland (4x)
- Bayou (2x)
- Swamp (basic, 2x)

This is a tracking issue for the LEA set's playability. Granular per-card status is tracked via per-card 'Card Compatibility' issues that reference this set issue.

Status legend:
- not-tested: card has not yet been instrumented
- partial: some abilities work, others don't
- working: all known behaviors verified
- broken: known game-breaking bug
