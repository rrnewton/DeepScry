## Fixes Applied

### 1. ✅ DFC (Double-Faced Card) Loading Support (e65e7ae)
- **Problem**: Ojer Axonil couldn't load (file named with both face names)
- **Fix**: Added prefix matching for card files (matches Java Forge behavior)
- **Files**: `src/loader/database_async.rs`

### 2. ✅ Infinite Priority Loop Prevention (42397ce)  
- **Problem**: Failed spell/ability/land attempts didn't switch priority
- **Fix**: Treat failures like passing priority to break infinite loops
- **Files**: `src/game/game_loop.rs`

### 3. ✅ Combat Damage Zone Tracking (e4f4de5)
- **Problem**: Creatures moved to graveyard twice during combat (double-move bug)
- **Fix**: Batch all creature deaths in single pass after combat damage
- **Files**: `src/game/actions.rs` (assign_combat_damage)
