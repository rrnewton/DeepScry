# Description

scripts/deploy-cloud.sh exits with "web_stage: unbound variable" at the end of a SUCCESSFUL deploy. Cause: function-local web_stage (mktemp -d, ~L433) referenced by a script-level EXIT trap under set -u. Harmless (deploy completes; web/ rsynced; probes pass) but noisy + leaves the temp dir. FIX: script-scope the var or guard the trap (rm -rf "${web_stage:-}"). Coordinate with parked trunk (origin c213c485) + mtg-590 which also edit deploy-cloud.sh.
