# Description

## Refactor Giant execute_effect() Function

## Problem

`execute_effect()` in `mtg-engine/src/game/actions/mod.rs` is a single giant
match statement handling every Effect variant inline. As of 2026-06-10
(branch claude/refactor-execute-effect, base integration 939ac4c1) it spanned
~3270 lines with ~78 match arms before decomposition began. Several arms inline
100-290 lines. This violates the <200-line-function / <2000-line-file
conventions (mod.rs is ~10,700 lines).

## Approach

Extract each effect FAMILY into a `game/actions/effects/<family>.rs` submodule
of `impl GameState` `execute_<effect>` methods; `execute_effect` becomes a thin
dispatcher that matches + delegates. PURELY STRUCTURAL / behavior-preserving:
each slice proven byte-identical against a fixed determinism baseline
(debug/refactor_baseline.sh: 7 deck pairs x 2 controllers x 4 seeds = 56 games,
[GAMELOG] stream byte-identical) PLUS a full capped `make validate`.

## Effect families (decomposition map)

- damage: DealDamage, DealDamageDivided, DealDamageDynamic*,
  DealDamageToTriggeredPlayer*, EachDamage, DamageAll, DealDamageXPaid*,
  PreventDamage, PreventDamageFromSource, PreventAllCombatDamageThisTurn
  (* = unresolved-fallback one-liners kept inline at the dispatch site)
- life: GainLife, GainLifeDynamic, LoseLife, SetLife, DrainMana
- cards/draw-discard: DrawCards, DiscardCards, Mill, Scry, Surveil, Loot, *XPaid
- zones: Destroy/Exile/Return/Sacrifice/Balance/Search/Dig/ChangeZoneAll/Move
- counters: PutCounter, MultiplyCounter, PutCounterAll, Proliferate, RemoveCounter
- tokens/copy: CreateToken, CopyPermanent, CopySpellAbility, Clone
- control/stat-mod: GainControl, Fight, Tap/Untap*, Pump*, Debuff, Animate,
  SetBasePowerToughness, GrantCantBeBlocked, Regenerate, AttachEquipment
- mana: AddMana, ChooseColor
- turn/phase: AddTurn, AddPhase
- stack/counter-magic: CounterSpell, ConditionalSelfCounter, UnlessCostWrapper,
  ModalChoice, ImmediateTrigger, CreateDelayedTrigger
- bending (Avatar set): Airbend, Earthbend, Firebend
- misc/control-flow: ClearRemembered, ClassLevelUp, Unimplemented, NoOp
