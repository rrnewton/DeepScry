# Description

Part of: mtg-dba689

Make pane borders grey instead of white to reduce visual clutter.

## Current behavior

All pane borders are white (default terminal color), which can be visually noisy.

## Target behavior

- Unfocused panes: Grey borders (`Color::Gray` or `Color::DarkGray`)
- Focused pane: White border (when pane focus is implemented)

## Implementation

For each panel/block creation:
```rust
let block = Block::default()
    .borders(Borders::ALL)
    .title("Panel Title")
    .border_style(Style::default().fg(Color::Gray));  // Add this
```

Affected methods in `fancy_tui_controller.rs`:
- `draw_left_tabs`: Info panel border
- `draw_bottom_left_tabs`: Actions panel border
- `draw_battlefield`: Battlefield panel borders
- `draw_card_details`: Card Details panel border
- `draw_hand`: Hand panel border

## Future work

When pane focus is implemented (mtg-b3f1fe), focused pane should use white border:
```rust
let border_color = if self.state.focused_pane == PaneId::Hand {
    Color::White
} else {
    Color::Gray
};
```
