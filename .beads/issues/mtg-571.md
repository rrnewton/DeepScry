### HASH UNIFICATION — DONE (this session, follow-up commit on PR #10)

- NEW mtg-engine/src/asset_hash.rs: `asset_hash_hex(bytes) -> String` = first 16 hex chars of blake3 digest. ASSET_HASH_HEX_LEN const. 4 unit tests (determinism, distinctness, known blake3("") vector af1349b9f5f9a1a6, lowercase-hex).

- Exporter (main.rs run_export_wasm) now calls asset_hash_hex; the old content_hash_hex (std DefaultHasher/SipHash) helper DELETED.
- NEW `mtg hash-asset <file>` subcommand prints asset_hash_hex of the file's bytes.
- scripts/hash_web_assets.sh now shells out to `mtg hash-asset` (locates target/release|debug/mtg or $MTG_BIN); the `sha256sum | cut -c1-16` line DELETED. No second hash impl anywhere.
