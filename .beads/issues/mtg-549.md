Test puzzle: test_puzzles/terror_destroys_creature.pzl

  {1}{B} Instant
  Destroy target nonartifact, nonblack creature. It can't be regenerated.

Script:
  A:SP$ Destroy | ValidTgts$ Creature.nonArtifact+nonBlack | NoRegen$ True

Findings (2026-05-29_#2461(53f1d817), compat-oldschool-wave6):

1. [x] Parses as {1}{B} Instant (mana_cost.generic=1, black=1).
2. [x] SP$ Destroy ability present with
       ValidTgts$ Creature.nonArtifact+nonBlack and NoRegen$ True (both
       preserved by the parser — verified by unit test).
3. [x] Runtime: casts, targets a nonblack creature, destroys it, moves it
       to the graveyard. Verified via game log below.

were already supported (same Destroy infra as Disenchant/Sinkhole). This
wave added regression tests + game-log evidence.
