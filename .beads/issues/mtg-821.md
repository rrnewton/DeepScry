# Description

TriggerEvent::CardDiscarded does not exist in the engine. T:Mode$ Discarded is parsed by parse_triggers() but falls through to Unknown("Discarded") with no registered handler.

Root cause: StaticAbilityMode::parse() returns Unknown("Discarded") since "Discarded" is not in the match arms. The effect_converter logs "Unknown Mode$ Discarded in SVar: ..." and silently skips it. No trigger fires.

Affected card: Monument to Endurance (and any card with Mode$ Discarded trigger)
Monument to Endurance script: T:Mode$ Discarded | ValidCard$ Card.YouOwn | TriggerZones$ Battlefield | Execute$ TrigCharm | TriggerDescription$ Whenever you discard a card, ...

Fix needed:
1. Add TriggerEvent::CardDiscarded to core/effects.rs
2. Add "Discarded" to StaticAbilityMode enum in loader/svar_parser.rs (or handle it directly in parse_triggers() in loader/card.rs)
3. Add parse_triggers() branch for Mode$ Discarded → TriggerEvent::CardDiscarded
4. Add check_triggers(TriggerEvent::CardDiscarded, ...) call in the discard code path in game/actions/mod.rs

Reproducer:
```sh
./target/release/mtg tui --p1 zero --p2 zero --p1-draw "Monument to Endurance;Island;Island;Island;Island;Island;Island" --p2-draw "Island;Island;Island;Island;Island;Island;Island" --seed 42 --verbosity 3 decks/championship/2025/01_manfield_izzet_lessons.dck decks/championship/2025/01_manfield_izzet_lessons.dck 2>&1 | grep -E "discard|Discard|Trigger|draw|Treasure|life"
```

Expected: when a card is discarded while Monument to Endurance is on battlefield, a Charm triggers offering: Draw a card / Create a Treasure / Each opponent loses 3 life.
Actual: No trigger fires when cards are discarded.

Findings (2026-06-05_#3008(50175e06)):
- StaticAbilityMode has no "Discarded" variant
- TriggerEvent has no CardDiscarded variant
- parse_triggers() has no Mode$ Discarded branch
- discard code path does not call check_triggers for discard events

CARD STATUS: BROKEN (engine feature gap)

## Fix (2026-06-06_#3013(41b11143)):
Added TriggerEvent::CardDiscarded and check_card_discarded_triggers() to actions/mod.rs. Fires from permanents on battlefield when controller discards. Also added to heuristic_controller.rs exhaustive match.

CARD STATUS: FIXED
