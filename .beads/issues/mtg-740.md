# Description

1994 World Championship compat (mtg-684 / mtg-713 B1). Aladdin: "{1}{R}{R}, {T}: Gain control of target artifact for as long as you control Aladdin."
Script: A:AB$ GainControl | Cost$ 1 R R T | ValidTgts$ Artifact | LoseControl$ LeavesPlay,LoseControl

ROOT CAUSE (was BROKEN — ability never offered):
- get_valid_targets_for_ability (targeting.rs) had NO GainControl arm (only the spell path did) -> activated GainControl enumerated zero targets.
- The activated-ability placeholder-rewrite match in game_loop/priority.rs had NO GainControl arm -> target + new_controller stayed placeholders and execute_effect bailed.
- Effect::GainControl carried no ValidTgts restriction and only a binary until_eot duration (and even until_eot revert was an unimplemented TODO mtg-77 — all one-shot GainControl was permanent).

FIX (general control-duration infra; CR 614 / 800.4a):
- New ControlDuration enum (Permanent / EndOfTurn / WhileControlSource) + a TargetRestriction on Effect::GainControl; converter parses LoseControl$ + ValidTgts$.
- Source threaded into the effect (resolve_gain_control_source, mirroring resolve_self_target) on both spell + activated paths.
- New Card.control_grant (source, grantee) + a self-correcting recompute_source_control SBA pass (sibling of recompute_aura_control) that reverts control to the owner the moment the grantee stops controlling the source. control_grant cleared when the card leaves the battlefield (CR 400.7).
- GainControl arms added to get_valid_targets_for_ability + the priority.rs activation rewrite.
