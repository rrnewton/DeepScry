# Description

Test all behavioral aspects of Library of Alexandria in MTG Forge-rs for the 1994 Old School playtest goal.

Card file: cardsfolder/l/library_of_alexandria.txt
Deck: 06_troll_disk_daniellebrunazzo (mtg-562)
Test puzzles: test_puzzles/library_of_alexandria_seven.pzl, test_puzzles/library_of_alexandria_six.pzl

Card text:
  Land
  {T}: Add {C}.
  {T}: Draw a card. Activate only if you have exactly seven cards in hand.

Script:
  A:AB$ Mana | Cost$ T | Produced$ C
  A:AB$ Draw | Cost$ T | PresentZone$ Hand | IsPresent$ Card.YouOwn | PresentCompare$ EQ7

Findings (2026-05-29_#2470(be2f61b4), wave10):

1. [x] Parses as Land with two {T} activated abilities (mana + draw).
2. [x] Mana ability {T}: Add {C} works.
3. [BROKEN -> FIXED] The draw ability's "Activate only if you have exactly
   seven cards in hand" restriction was silently dropped: it could be
   activated at ANY hand size. Root cause: parse_activated_abilities did not
   parse `IsPresent$ | PresentZone$ | PresentCompare$` into any activation
   restriction, and can_activate had no gate for it.
4. [x] Fix: new core::ActivationCondition { filter, zone, op, count } +
   core::CompareOp (EQ/GE/LE/GT/LT). Parsed in
   CardDefinition::parse_activation_condition; enforced in
   push_activatable_abilities/can_activate by counting cards via the existing
   GameState::count_cards_matching_filter (extended to handle Card/Permanent
   wildcards, Land/Artifact/Creature/Enchantment card types, and color quals).

GENERALIZED to the whole activation-gate bug class: 29 activated abilities use
`IsPresent$ + PresentCompare$` (Magus of the Library EQ7, Cryptic Caves GE5
lands, Mistveil Plains GE2 white permanents, Madblind Mountain, Moonring
Island, Underhanded Designs, ...). All now honour the gate.

Reproducer:

```sh
## Draw IS available with exactly 7 cards in hand:
./target/release/mtg tui --start-state test_puzzles/library_of_alexandria_seven.pzl \
  --p1=fixed --p2=zero --p1-fixed-inputs="activate Library of Alexandria;*" \
  --seed 42 --verbosity 3
## Draw is NOT available with 6 cards (command fails to match any action):
./target/release/mtg tui --start-state test_puzzles/library_of_alexandria_six.pzl \
  --p1=fixed --p2=zero --p1-fixed-inputs="activate Library of Alexandria;*" \
  --seed 42 --verbosity 3
```

Expected log evidence:

```
## seven.pzl:
Library of Alexandria activates ability: Draw a card. Activate only if you have exactly seven cards in hand.
Player 1 draws ...
## six.pzl:
Error: InvalidAction("... 'activate Library of Alexandria' did not match any available action ...")
```
