# Description

## Summary

Tournament testing with old_school decks revealed **228 errors** across tournament games when running Heuristic vs Random controller games.

**Status**: ALL MAJOR ERROR CATEGORIES FIXED! ✅

## Error Categories

### 1. ✅ FIXED: Infinite Loop with Activated Abilities (39.5% of errors)
**Error**: "Priority round exceeded max actions (1000), possible infinite loop"  
**Count**: 90 errors (39.5% of all errors)  
**Status**: ✅ **FIXED in commit c144036**

**Root Cause**: Activated abilities with mana costs didn't auto-tap lands before paying costs.

**Fix**: Added auto-tap logic before paying ability costs, mirroring spell casting behavior.

### 2. ✅ FIXED: Counterspell Fizzle Logic (13.2% of errors)
**Error**: "Cannot counter a spell that is not on the stack"  
**Count**: 30 errors (13.2% of all errors)  
**Status**: ✅ **FIXED in commit fb47d60**

**Root Cause**: When multiple counterspells targeted the same spell, the second one would error instead of fizzling per MTG Rules 608.2b.

**Fix**: Extended target validation in resolve_spell() to check if counterspell targets are still on stack, causing spell to fizzle if not.

**Verified Fixed**:
- Seed 5582354775641719354 ✓
- Seed 12887778920775399169 ✓  
- Seed 16866519314934403830 ✓

### 3. ✅ FIXED: Card Zone Tracking Errors in Combat (47.4% of errors)
**Error**: "Card X not found in source zone" (various card IDs)  
**Count**: 108 errors (47.4% of all errors)  
**Status**: ✅ **FIXED in commit e4f4de5**

**Root Cause**: During combat damage, creatures were being moved to graveyard twice:
1. deal_damage_to_creature() moved creatures with lethal damage immediately
2. Deathtouch logic tried to move deathtouch-damaged creatures after all damage

This caused the second move_card() call to fail because creatures were already in graveyard.

**Fix**: Refactored assign_combat_damage() to:
- Collect all creatures that should die (lethal damage OR deathtouch)
- Move all dying creatures to graveyard in a single pass
- Matches MTG Rules 704.5f (state-based actions)

**Verified Fixed**:

- Seed 11932290975579144432 ✓
- Seed 7011722551026603242 ✓
- All 40 reproducers from monored.dck tournament ✓
