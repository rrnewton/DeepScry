# Description

## Summary

Tournament testing with old_school decks reveals a **4.1% error rate** (120 errors in 2,916 games) when running Random vs Random controller games.

## Test Configuration

- **Decks**: 6 old_school decks from `decks/old_school/`
- **Duration**: 3 seconds
- **Games played**: 2,916 games
- **Games per second**: 969
- **Controllers**: Random vs Random
- **Errors found**: 120 (4.1% failure rate)

## Error Categories

### 1. Card Zone Tracking Errors (~40% of errors)
**Error**: "Card X not found in source zone"

This indicates bugs in card tracking when moving cards between zones (battlefield, hand, graveyard, exile).

**Example reproduction command**:
```bash
cargo run --release --bin mtg -- tui --p1 random --p2 random --seed 4819709227262051603 \
  "decks/old_school/06_troll_disk_daniellebrunazzo.dck" \
  "decks/old_school/01_rogue_rogerbrand.dck"
```

**Verified**: Error reproduces exactly with the command above, failing when Swords to Plowshares tries to target a card.

### 2. Counterspell Targeting Errors (~60% of errors)
**Error**: "Cannot counter a spell that is not on the stack"

This indicates bugs in counterspell timing/targeting logic - the game is trying to counter spells that have already resolved or were never on the stack.

**Example reproduction command**:
```bash
cargo run --release --bin mtg -- tui --p1 random --p2 random --seed 1264870707092871110 \
  "decks/old_school/02_thedeck_peterschnidrig.dck" \
  "decks/old_school/02_thedeck_peterschnidrig.dck"
```

## Reproduction System

All 120 errors are saved in `tournament_errors.txt` with:
- Game index
- Exact error message
- Deck paths
- All seeds (game seed, P1 seed, P2 seed)
- Complete reproduction command

Each error can be precisely reproduced with a single command.

## Next Steps

1. Investigate card zone tracking logic (Swords to Plowshares interaction)
2. Investigate counterspell timing/targeting validation
3. Add regression tests for fixed issues
4. Re-run tournament to measure error rate reduction
