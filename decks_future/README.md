# Future Decks

This directory contains deck files that use cards not yet available in the forge-java card database.

These decks are excluded from automated testing (determinism tests, deck loading tests) because they reference cards that don't exist in our database yet. They are preserved here for future use once the card database is updated.

## Contents

- `mtg_world_2025/` - Magic: The Gathering World Championship 31 (2025) Top 8 decklists
  - Uses cards from the Avatar: The Last Airbender Standard set and other recent sets
  - Will become usable once forge-java adds support for these sets
